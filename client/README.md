# E-Commerce Client

React frontend for the E-commerce platform.

## Setup

1.  **Navigate to the client directory**:
    ```bash
    cd client
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Environment Variables**:
    Check `.env` and update `VITE_API_URL` if your backend is running on a different port.

## Running

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Production Build

```bash
npm run build
npm run preview
```
